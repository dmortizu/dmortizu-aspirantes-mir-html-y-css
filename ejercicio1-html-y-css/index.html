Receta: Ensalada

Ingredientes

Los ingredientes necesarios para preparar la ensalada son los siguientes:

  • Pechuga
  • Lechuga
  • Tomate
  • Cebolla
  • Maiz

Preparación

Se pican todos los ingredientes, se mezclan y listo!

Bonus: Nueva preparación de ensaladas

Ingredientes

Los ingredientes necesarios para preparar la nueva ensalada son los siguientes:

  1. Cerdo
  2. Cebolla cabezona morada
  3. Tomate pinto
  4. Aguacate
  5. Aceite de oliva

Preparación

Se pican todos los ingredientes, el aguacate se tritura hasta quedar blando, se mezclan junto con el aceite de oliva y listo!
